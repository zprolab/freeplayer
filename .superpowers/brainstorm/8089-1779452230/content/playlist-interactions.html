播放列表管理交互

创建列表 · 添加曲目 · 右键操作 · 对接现有设计系统

1. 创建新播放列表 — 轻量弹窗

点击侧边栏 Playlists 标题旁的 "+" 按钮，弹出简洁的创建对话框。与现有 Edit Modal 风格一致（复用 CSS 变量）。

Create Playlist 弹窗
New Playlist ✕
Cancel Create

2. 添加曲目到播放列表 — 右键菜单

在 Library 表格中右键曲目，上下文菜单增加"Add to Playlist"选项，hover 展开子菜单显示所有用户播放列表。

右键菜单 — 添加至播放列表
✏️ Edit Metadata
➕ Add to Playlist ▶
🗑️ Delete Track
🎸 Favorites
🏃 Workout
🌙 Chilled
✨ New Playlist...

交互规则：

• 已在该列表中的曲目显示 ✓ 标记

• "New Playlist..." → 弹出创建弹窗 → 创建后自动添加

• 重复添加同一曲目到同一列表时忽略

3. 播放列表右键操作

在侧边栏右键播放列表名，弹出管理菜单。

播放列表右键菜单
▶️ Play All
✏️ Rename
🗑️ Delete

• "All Tracks"（主列表）不显示右键菜单

• Rename → 内联编辑 / 小弹窗

• Delete → 确认对话框（参考现有 Reset Database）

4. 播放列表选中态 — 集成到 Library 视图

选中列表后，切换到 Library 视图，顶部标题区增强显示列表信息。
