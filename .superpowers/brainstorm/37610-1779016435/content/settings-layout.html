Key Design Decisions

  • Radio cards for import mode — makes the trade-off visible at a glance (disk space vs portability)
  • Segmented button for visualizer default — compact and matches the visualizer UI in Now Playing
  • Red-bordered danger zone — visually separates destructive actions, requires confirmation dialog
  • Path display uses monospace font inline, consistent with the rest of the app
  • Settings are saved immediately on change (no "Save" button needed)
